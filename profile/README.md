### Mini Vue Admin
坚持Less is More，一个基于Vue + Element Plus的模板框架，采用JS开发，精简的依赖和组件，适合新手入门，也适合高手拓展。  

并且提供Python、Java、Kotlin语言和框架的后端脚手架。
